Guide

Read Me

This extension injects a toolkit into OPCloud Sandbox so you can bootstrap runtime access, export the current OPD tree to JSON, and import a previously exported tree back into the page.

OPCloud itself remains the real editor and renderer. The toolkit tries to work through OPCloud's runtime model instead of redrawing diagrams separately.

Project page: github.com/ASSANDHOLE/opcloud_toolkit

Basic Workflow

  1. Open https://opcloud-sandbox.web.app/.
  2. Click Initialize Toolkit once for the current tab.
  3. Use Download JSON or Copy JSON to export.
  4. Use Choose JSON File or paste JSON to import.

Limitations

  • OvertimeException links are not tested.
  • No max-duration handling is currently defined or enforced.
  • Fonts are not explicitly handled during export/import fidelity.
  • In-diagram in-zoom navigation/UI behavior is not implemented as a popup feature.
  • Some advanced or fancy OPCloud functions are not covered by this toolkit yet.
  • Link geometry and state positions can still be imperfect after reconstruction.

Possible Breakage

  • This toolkit depends on OPCloud Sandbox runtime internals and reverse-engineered behavior.
  • Upstream OPCloud updates may break bootstrap, import/export, or geometry reconstruction.
  • If something suddenly stops working, first reload the extension and refresh the OPCloud page.

Practical Advice

  • Keep backups of important JSON exports.
  • Test imports on a disposable page first when trying a new diagram shape or feature.
  • If an import looks wrong, export again and compare the structure before editing further.
